    Reader is an open source, visible and invisible UI wrapper, for Microsoft speech synthesis API.

    Project maintained by BoazJR Hosted on GitHub Pages — Theme by mattgraham

    Aloha all !!

    This Windows program will try to read any text you have copied. There are two ways to use the program, an invisible window and a visible window.

    The Visible Window: Simply select the text, click CTRL+C (to copy the text) and click Play.

    The Invisible Window: Again select the text, click CTRL+C and double click the lips icon in your task bar. To access the settings place your mouse on the icon on the taskbar and wait for the preview window to open where you can click the settings button.

    The Settings Window: click the settings icon to access the settings window where you can change the speed of the reader, add regular expressions pattern to replace patterns of text with words of your choosing right, and switch between the visible and the invisible window.
